\subsection{Detailed Description}
Statistics about a text and the recognition process. 

This class stores a number of statistic data regarding the recognition process and the text produced.

First, it store the elapsed time on every stage, accumulated internally in the critical processes. Then, it computes the number of words in the text and the appearance rate of every single word.
